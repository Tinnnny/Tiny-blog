# 神经网络模型论文摘要
虽然作为国内电力仪表行业的领军企业集团，但是现在华立电力仪表的数字化生产仍然是不完全的，质量管理作为智能制造的重要环节，是电力仪表生产的关键问题之一。目前华立以及国内其他厂家并没有针对电力仪表的质量管理系统，这对于我国电力仪表制造行业的长远发展是十分不利的。在实际生产中，评价一个电表的重要指标是电表的准确度等级，但这并不能完全一个电表的质量。

查阅相关文献得知，目前，有几种模型和算法可以综合评价供应商生产的智能电表的质量。这些模型和算法主要可以分为三类:
1. 根据一定的指标对智能电表和供应商进行评价。
2. 从不同方面提出智能电表的多项指标，并进行处理数据采用简单相加加权(SAW)法对设备进行评价。然而，不同的指标对智能电表的运行质量有不同的影响，因此进行评价如果不考虑索引的重要性，那么这种算法的结果是不准确的。
3. 应用专家评分算法、层次分析法(AHP)等主观赋权法直接确定各指标的权重，其中考虑了指标对智能电表影响的差异性。然而，结果是相当主观的，由于专家的知识和经验的限制，这可能会导致很大的差异不同专家对智能电表运行质量的评价结果。

对于电表质量的评估，最有发言权的是电表专家。不同专家应用层次分析法等方法对电表各个指标进行权重的赋值很难得到统一，但不同专家对电表的质量认知却是相同的。因此本文利用神经网络作为电表质量模型，将电表质量分为五等，其中一等最佳，不合格电表为五等，训练样本的等级根据专家给定。最终训练完成的神经网络便能用于判断电表的质量等级。BP神经网络是一种多层的前馈神经网络，其主要的特点是：信号是前向传播的，而误差是反向传播的。
由于电表型号的差异性，针对三相电表本课题选用16个输入层神经元、1个隐层、5个隐层神经元、 5个输出层神经元，输入层和隐含层之间的传递函数为型的正切函数。其中的指标分别为：0.5L,IB,检误差、检日计时误差值范围、1.0,IB,检误差、光电脉冲检测、IB脉冲检测、检测日计时、1.0 Imax、0.5L Imax、1.0 100%lb、0.5L 100%lb、0.5L 10%lb、1.0 5%lb以及PCB板元器件焊接质量。除PCB板元器件焊接质量外，其余数据均可在产线上获取。PCB板元器件焊接质量的评估采用图像处理的方式，将测试板与标准板进行比较，利用它们之间差异性的大小来评估测试板的元器件焊接质量，焊接质量同样分为5个等级，从1~5级PCB板元器件焊接质量逐级下降。

# 论文摘要
## 基于灰色关联度的智能电能表性能评价方法研究
为解决传统智能电能表供应商综合评价中主观意识强、数据繁杂等问题，提出了熵权理论和灰色关联度相融合的综合评价方法。该方法从产品质量的角度出发，将其划分为生产监造、供货前质量评估、到货后质量评估、现场运行评估四个阶段，并通过搭建电能表全寿命周期管理云平台系统对电能表的设计、关键元器件、生产、监造、检测、运行、报废鉴定等各环节的相关数据进行提炼，得到可以量化的数据指标，作为智能电能表综合评价的主要依据，以此为基础建立评价指标体系和多目标综合评价模型，实现从多层次、多角度对电能表供应商综合性能水平作出定量评价，其评价结果客观、全面、准确。
